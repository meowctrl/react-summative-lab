🚀How To Get Started:
Install!
First run 'npm install' for needed dependencies,
Next run npm run server to start the backend, make sure you're in the right directory and run 'npm run dev'
Lastly click on the localhost link provided, or navigate to http://localhost:5173 in a separate browser for the intended site to display!
